# 322.BOOTH // DAW

Web-based DAW with real-time pitch correction, built-in Web Audio plugins, and VST3 bridge support.

Part of the [322.studio](https://322.studio) ecosystem.

## Features
- 6 built-in plugins (EQ, Compressor, Reverb, Delay, Distortion, Chorus)
- Real-time pitch correction via AudioWorklet + YIN algorithm
- VST3 bridge for desktop power users
- 322.studio visual identity

## Stack
React + Vite + TypeScript + Web Audio API
